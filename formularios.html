¿Qué son los formularios?

Los formularios son uno de los principales puntos de interacción entre usuario y una página web o aplicación.


Su principal función es permitir a los usuarios el ingreso de datos y que la aplicación de pueda trabajar con los mismos, sea por ejemplo, para guardar dicha información en una base de datos, utilizarla en una búsqueda o simplemente realizar cambios en la GUI a partir de la selección del usuario.


Las etiquetas básicas que puede contener un formulario son:

  • <form> </form>: apertura y cierre de formulario
    • action: permite indicar la acción que realizará el formulario cuando se quiera enviar
    • method: indica el método mediante el cual se enviará el formulario (ej: POST)
  • <label> </label>: Apertura y cierre para una etiqueta de un elemento dentro del formulario, en otras palabras, el texto que se lleva al frente
    • for: se utiliza para indicar a qué input hará referencia la etiqueta
  • <input> </input>: Se utiliza para permitir cuadros de texto donde el usuario ingrese datos. Algunos complementos de esto son:
    • type: permite indicar el tipo de entrada que permitiremos, por ejemplo: email, text, password, etc.
    • id: permite asignar una id (identificación/nombre) que sea única para cada input
  • <textarea> </textarea>: Se utiliza para cuadros grandes de texto, como los típicos de observaciones
    • id: permite asignar una id (identificación/nombre) que sea única para cada textarea
